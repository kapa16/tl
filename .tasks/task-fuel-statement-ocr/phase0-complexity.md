# Phase 0: Инициализация

## Оценка сложности

**Критичная / Сложная** (верхняя граница)

Причины:
- Новая внешняя утилита OCR (в репозитории нет исходников, только паттерн QR)
- Три разных физических бланка при одном MXL-макете печати в CF
- Распознавание рукописных цифр в шаблонных «гребёнках» — компьютерное зрение, не готовый API 1С
- Интеграция с существующим EPF и документом `GDS_ЗаправочнаяВедомость`
- Расхождения QR печати vs QR в EPF, pre-created документы
