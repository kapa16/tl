Форма "Настройки мониторинга"

 

Закладка "1С:ЦСМ". Позволяет включить использование механизмов взаимодействия с системой мониторинга "ЦСМ".

  • Использовать ЦСМ. Включает использование взаимодействия с системой мониторинга "ЦСМ".

Закладка "Omnicomm". Позволяет включить использование механизмов взаимодействия с системой мониторинга "Omnicomm".

  • Использовать Omnicomm. Включает использование взаимодействия с системой мониторинга "Omnicomm".
  • Хранение загруженных данных с интервалом. Например, при выборе значения "15 минут" в процессе загрузки данных будет сформирована запись на каждый 15-минутный интервал. Чем больше записей в регистре сведений, тем точнее можно сопоставить эти записи с путевыми листами в отчетах, однако большое количество записей приведет к росту объема информационной базы. Рекомендуется использовать стандартное значение: 15 минут.
  • Хранение координат с интервалом. Интервал хранения координат (в секундах). Позволяет сократить объем информации, загружаемой из системы мониторинга.
  • Параметры хранения данных и регламентного задания "Загрузка данных".

  • Загружать время работы двигателя. Признак загрузки данных по времени работы двигателя. Доступно для интервалов 15 минут, 30 минут, 1 час, 12 часов и 24 часа.

Закладка "Wialon". Позволяет включить использование механизмов взаимодействия с системой мониторинга "Wialon".

  • Использовать Wialon. Включает использование взаимодействия с системой мониторинга "Wialon".
  • Хранение загруженных данных с интервалом. Например, при выборе значения "15 минут" в процессе загрузки данных будет сформирована запись на каждый 15-минутный интервал. Чем больше записей в регистре сведений, тем точнее можно сопоставить эти записи с путевыми листами в отчетах, однако большое количество записей приведет к росту объема информационной базы. Рекомендуется использовать стандартное значение: 15 мину.
  • Хранение координат с интервалом. Интервал хранения координат (в секундах). Позволяет сократить объем информации, загружаемой из системы мониторинга.
  • Использовать смены. При использовании смен будет сформирован один запрос к системе мониторинга с настроенными сменами. Если смены не используются - для получения детальных данных будет выполнено несколько запросов с шагом, указанным в настройке "Хранение загруженных данных с интервалом".
  • Пробег. Настройка определяет колонку таблицы "Сводка" в системе Wialon, из которой будет загружен пробег: "Пробег в поездках" или "Пробег по всем сообщениям".
  • Источник координат. Настройка определяет таблицу в системе Wialon, из которой загружаются координаты: "Трассировка сообщений" или "Трассировка датчиков".
  • Время жизни ссылки на локатор. Срок действия ссылки на локатор. Значение может быть указано в минутах, часах или днях (выбор из выпадающего списка). В качестве срока действия можно указать любое значение (от 1 до 1000) или сделать его неограниченным (ввести 0).
  • Параметры хранения данных и регламентного задания "Загрузка данных".

Параметры хранения данных и регламентного задания "Загрузка данных":

  • Загрузить пробег и расход ГСМ. При работе регламентного задания будет загружен пробег и расход топлива по указанным в таблице ТС транспортным средствам.
  • Загрузить события по мониторингу. При работе регламентного задания будут загружены дополнительные сведения по указанным в таблице ТС транспортным средствам.
  • Загрузить координаты. При работе регламентного задания будет загружена история местоположения по указанным в таблице ТС транспортным средствам.
  • Загрузка данных регламентным заданием за. Например, при выборе значения "24 часа" регламентное задание выполнит загрузку данных за последние 24 часа с момента начала выполнения регламентного задания.
  • Таблица ТС. Позволяет указать список транспортных средств, по которым необходимо загружать данные регламентным заданием.
