Обработка "Загрузка данных Omnicomm"

Назначение

Эта обработка позволяет производить загрузку данных из системы спутникового мониторинга Omnicomm за указанный промежуток времени. При загрузке данных используется индивидуальная настройка учетной записи системы мониторинга, указанная в каждом ТС.
Если учетная запись в ТС не указана, то настройки системы мониторинга берутся из формы "Настройки мониторинга".

Настройки обработки:

  • Загрузить пробег и расход ГСМ. Если флаг установлен, загружается история пробега и расхода топлива ТС. Данные записываются в регистр сведений "Пробег и расход по мониторингу".
  • Загрузить события по мониторингу. Если флаг установлен, загружается информация  о следующих видах событий: заправка ГСМ, слив ГСМ, мгновенное превышение скорости, долгосрочное превышение скорости, нарушение предельных значений датчиков. Данные записываются в регистр сведений "События по мониторингу".
  • Загрузить координаты. Если флаг установлен, загружается история местоположения ТС . Данные записываются в регистр сведений "Координаты по мониторингу".

Относится к подсистеме: «Мониторинг».

Формы

  • Форма. Основная форма. На форме расположены следующие элементы:
    1. Реквизит "Дата с" - тип Дата. Дата, с которой необходимо получить данные;
    2. Реквизит "Дата по" - тип Дата. Дата, до которой необходимо получить данные;
    3. Табличное поле "ТС". Содержит список ТС, для которых будет выполнена загрузка данных. Поддерживает как одиночный выбор ТС, так и множественный. При нажатии на кнопку "Заполнить" откроется форма подбора ТС.
