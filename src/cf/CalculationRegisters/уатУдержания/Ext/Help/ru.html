Регистр расчета "Удержания"

Назначение

В этом регистре хранятся данные по начислению удержаний по сотрудникам/водителям организаций.

Относится к подсистеме: «Управление персоналом».

Измерения

  • Организация. Тип значения: справочник «Организации». Организация, по которой вводится удержание. 
  • Подразделение. Тип значения: справочник «Подразделения организаций». Подразделение организации, по которому вводится удержание. 
  • Сотрудник. Тип значения: справочник «Сотрудники (УАТ)». Сотрудник, по которому вводится удержание.
  • Тариф. Тип значения: справочник «Тарифы зарплаты». Тариф зарплаты с применением "Удержание", который использован для ввода удержания. 
  • ТС. Тип значения: справочник «Транспортные средства и оборудования». ТС, с которым связано удержание. 

Ресурсы

  • Результат. Тип значения: «Число». Сумма удержания в валюте управленческого учета. 

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом. 

Документы, порождающие движения: «Начисление удержаний».
