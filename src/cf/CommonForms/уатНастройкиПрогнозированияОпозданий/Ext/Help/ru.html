Форма "Настройки прогнозирования опозданий"

Маршрутные листы

Прогнозирование опозданий выполняется для проведенных Маршрутных листов, которые отправлены, но не выполнены. Маршрутные листы, по которым плановая дата выполнения больше 1 месяца назад, считаются неактуальными, и прогнозирование для них не выполняется.

Алгоритм прогнозирования

1. Для каждого Маршрутного листа определяется последний выполненный пункт.

  • Будет прогнозироваться прибытие в следующий за ним пункт, без учета возможного прохождения пунктов Маршрутного листа не по порядку. Опоздания в дальнейшие пункты не прогнозируются.
  • В случае, если в Маршрутных листах ведется учет планового и фактического времени, и для пункта заполнено фактическое время прибытия, то пункт считается пройденным, и рассчитывается прибытие в следующий за ним пункт, даже если флаг "Выполнен" для пункта не установлен.

2. Для каждого Маршрутного листа определяется текущее местоположение ТС - по имеющимся в программе данным из систем спутникового мониторинга, мобильного приложения водителя, или по фактическим данным прибытия в пункт из Маршрутного листа.

  • В случае, если данные и текущем местоположении ТС старше 12 часов, то они считаются неактуальными, и прогнозирование опоздания по этому Маршрутному листу не выполняется.
  • Данные о местоположении ТС из маршрутного листа считаются вспомогательными, используются только даты фактического прибытия и убытия, при их наличии, в последний выполненный пункт.

3. Рассчитывается плановое время прибытия в следующий пункт. При этом:

  • Не учитывается время стоянки и ожидания, указанное в Маршрутном листе. Расчет выполняется исходя из предположения, что в момент прогнозирования ТС находится в движении в следующий пункт. 
  • Учитываются данные регистра "Средняя скорость движения", при их наличии

Связанные объекты

Для просмотра данных об опозданиях рекомендуется использовать отчеты "История опозданий", "Текущие опоздания" и "Показатели выполнения Маршрутных листов".

Данные об опозданиях сохраняются в регистрах "Текущие прогнозы опозданий" и "История прогнозов опозданий".
