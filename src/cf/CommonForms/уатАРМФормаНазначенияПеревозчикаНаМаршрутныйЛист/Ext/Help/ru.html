Форма "Назначение перевозчика на маршрутный лист"

Форма предназначена для использования в 2 сценариях:
1. Запуск из АРМ Логиста - назначение перевозчиков и договоров для всех выбранных Маршрутных листов. В этом случае необходимо, чтобы все Маршрутные листы находились в рамках одного документа "Планирование работы ТС".
2. Запуск из формы Маршрутного листа - назначение перевозчика, договоров и ТС для одного Маршрутного листа.

Документ "Планирование работы ТС" с планами для перевозчиков определяется, исходя из организации и даты отправления Маршрутного листа (или даты закрытия, если он закрыт).
Плановая выработка определяется по документу "Планирование работы ТС".
Фактическая выработка определяется по регистру "Объемы перевозок", записи в котором создаются Маршрутными листами.

В случае, если автоматическое распределение не назначило перевозчика для какого-то Маршрутного листа, рекомендуется проверить, что в нем рассчитана выработка.

Отборы
При запуске формы установлены отборы по всем аналитикам, которые используются в Сценарии планирования. Возможно отключение этих отборов для просмотра запланированных объемов перевозок, не совпадающих по аналитикам с Маршрутным листом. В этом случае в таблице перевозчиков могут возникать строки, которые по набору аналитик выглядят одинаково, но относятся к разным строкам Планирования работы ТС и соответственно имеют разные плановые и фактические количества выработки.
