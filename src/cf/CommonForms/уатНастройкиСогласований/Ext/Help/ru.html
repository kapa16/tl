Настройки согласований

Форма предназначена для настройки согласований по организации или подразделению.

Группа исполнителей согласования по умолчанию: определяет пользователей, которые будут подставлены в согласование при его создании. После этого список пользователей может редактироваться вручную. Настраивается в разрезе организации и подразделения. Если настроены и группа по подразделению, и группа по организации, то группа по подразделению используется в приоритете.

Согласование по умолчанию: определяет, будет ли для новых документов автоматически установлен флаг "Требует согласования". Настраивается в разрезе организации.
