Общая форма "Подбор ЭПД"

Предназначена для создания документов: "Электронные транспортные накладные" и "Электронные заказы (заявки)".

Несколько Заказов на ТС возможно сгруппировать в один документ ЭПД, если в Заказах на ТС совпадает грузоотправитель, грузополучатель, рейс, адрес отправления и адрес прибытия.

В группировку не включаются заказы, по которым ранее был создан документ ЭПД.

Для каждой выбранной группы создается отдельный документ ЭПД. Группы автоматически сортируются по их номеру.
