Цены и валюта

Диалог "Цены и валюта" предназначен для заполнения и изменения данных об установке порядка оплаты. Открывается при нажатии кнопки "Редактировать цены и валюту" в форме документа.

При открытии диалога показываются установленные на текущий момент времени значения параметров. Количество и характер управляющих элементов в этом окне зависят от вида документа.

Это могут быть такие элементы, как:

  • Валюта документа. Группа реквизитов:
    • Валюта - валюта документа. Справочник "Валюты". Валюта документа. Соответствует валюте договора, указанного в документе. Доступно только для просмотра.

    • Курс - курс выбранной в документе валюты.

    • Кратность - кратность выбранной в документе валюты. Доступно только для просмотра.

  • Валюта упр. учета. Группа реквизитов:

    • Валюта - валюта управленченского учета. Справочник "Валюты". Доступно только для просмотра.

    • Курс - курс валюты управленченского учета. Доступно только для просмотра.

    • Кратность - кратность валюты управленченского учета. Доступно только для просмотра.

  • Налоги. Группа реквизитов:

    • Учитывать НДС. Признак учета НДС.

    • Сумма включает НДС. Возможны два варианта заполнения суммы НДС в документах. Если этот флажок установлен, то предполагается, что НДС уже включен в сумму документа. В графе "Всего" отображается сумма с включенным НДС. НДС вычисляется исходя из суммы по каждой позиции и ставки НДС, указанной в документе. Если флажок не установлен, то в сумму по документу (графа "Всего") не включается НДС.

    • НДС включать в стоимость. Если флажок установлен, то при формировании проводок НДС будет включен в стоимость.
