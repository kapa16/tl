План видов расчета «Удержания»

Содержит список удержаний, используемых при начислении удержаний. В этот план видов расчета включаются такие удержания сотрудников, как штрафы ГИБДД, возмещение ущерба и другие.

Реквизиты

  • Способ расчета. Для удержаний используется только один способ расчета - "Фиксированной суммой".
  • Код вида начисления для выгрузки. Следует указать код соответствующего вида удержания плана видов расчета "Удержания" типовой конфигурации (ЗУП, ERP).
  • Статья доходов. Используется для отражения удержаний по оперативным доходам компании.
