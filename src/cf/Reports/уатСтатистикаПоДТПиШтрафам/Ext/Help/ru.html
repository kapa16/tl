Отчет "Статистика по ДТП и штрафам"

Назначение

Данный отчет отображает статистику по количеству ДТП и штрафам. Сведения выводятся в виде группировок по ссылочным полям и в виде диаграммы с распределением по месяцам. В отчет попадают записи ДТП и Штрафов записанные на дату попадающую в период. Также за этот период должен быть пробег на ТС. По этой причине ТС с счетчиком моточасов в отчете не отображаются.

Относится к подсистеме "БДД".

Способы вызова

Вызывается из раздела "БДД" -> "Отчеты / Рейтинг водителей по ДТП".
