Отчет "Ведомость ТЭП"

Назначение

Данный отчет позволяет получить информацию о технико-эксплуатационных показателях.

Относится к подсистеме "Работа ТС".

Вариант "Ведомость ТЭП"

Отчет отображает детально по каждому ТС информацию за период. Показатели, выводимые в отчете:
  • Время в наряде, Время в работе, Время в ремонте, Перевезено, Грузооборот, Ездок с грузом, Пробег общий, Пробег с грузом, Пробег порожний, Расход топлива - по данным из регистров учета Выработка ТС и Фактический и нормативный расход ГСМ в ТС.
  • КПП (коэффициент порожнего пробега) - доля порожнего пробега от общего.
    Рассчитывается по формуле: КПП = Ппор/Побщ = (Побщ - Пгр)/Побщ, где Ппор - порожний пробег, Пгр - пробег с грузом, Побщ - общий пробег.
  • Автодней в хозяйстве - общее количество дней эксплуатации ТС за период отчета;
  • Автодней в работе - количество дней по Путевым листам при условии нахождения ТС в колонне, указанной в параметрах отчета;
  • Автодней в ремонте - количество дней по Ремонтным листам.
  • Автодней выходных - количество выходных дней по графику работы ТС за период отчета с учетом изменения местонахождения ТС (Ввод в эксплуатацию ТС, Перемещение ТС, Выбытие ТС).
  • КИП (коэффициент использования автопарка) - доля автодней в работе от автодней в хозяйстве.
    Рассчитывается по формуле: КИП = АДр/АДх, где АДр - количество автодней в работе, АДх - количество автодней в хозяйстве.
  • КТГ (коэффициент технической готовности автопарка) - показывает степень готовности автомобилей для выполнения перевозок.
    Рассчитывается по формуле: КТГ = АДи/АДх = (АДх - АДрем)/АДх, где АДи - количество автодней исправных автомобилей, АДрем - количество автодней в ремонте, АДх - количество автодней в хозяйстве.

Вариант "Сводные ТЭП"

Отчет отображает в свернутом виде по организациям информацию за период о показателях использования автопарка. Показатели, выводимые в отчете:
  • Автодней в хозяйстве, Автодней в работе, Автодней в ремонте, Автодней выходных, КИП, КТГ - аналогично варианту отчета "Ведомость ТЭП".
  • Календарных дней - общее количество календарных дней в периоде отчета
  • Пробег общий - пробег общий всех ТС за период отчета, автомобиле-километры.
  • Среднесписочное количество ТС - отражает среднее количество ТС, числящееся в автохозяйстве за период отчета. Определяется по формуле Сумма(Автодней в хозяйстве) / Календарных дней.
  • Средний пробег - усредненный пробег всех ТС в автохозяйстве за период отчета. Рассчитывается по формуле Пробег общий / Среднесписочное количество ТС.
  • КВЛ (коэффициент выхода на линию) - коэффициент использования автопарка без учета выходных дней.
    Рассчитывается по формуле: КВЛ = АДр/(АДх - АДвд), где АДр - количество автодней в работе, АДх - количество автодней в хозяйстве, АДвд - количество выходных автодней.

Примечания:

  1. В показатель Автодней в ремонте не входят дни, когда выписаны и Путевой и Ремонтный листы, в этом случае приоритетным считается Путевой лист и нахождение в ремонте не учитывается.
  2. В показателе КВЛ выходные автодни считаются по индивидуальному графику работы каждого ТС. Если график не указан, то все дни считаются рабочими.
  3. Итоги по показателям Средний пробег, КИП, КТГ, КВЛ рассчитываются по данным строки итогов, а не по итоговой сумме в группировке.
