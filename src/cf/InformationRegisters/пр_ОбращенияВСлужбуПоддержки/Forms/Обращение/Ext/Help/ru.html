Обращение в службу поддержки

Форма используется для отправки сообщения об ошибке, пожелания или вопроса, связанных с работой программы, в службу поддержки программного продукта.

К форме можно перейти из раздела Главное – Обращение в техническую поддержку.

Для отправки пожелания в форме должны быть заполнены реквизиты:

  • Тема обращения;
  • Автор;
  • Номер телефона;
  • Электронная почта;
  • Роль (должность) пользователя – роль (должность), в которой автор обращения выполняет действие в программе. Например, менеджер по продажам/специалист по технической поддержке/бухгалтер.
  • Какая задача решается и зачем – подробное описание решаемой бизнес-задачи и ее ценность для организации.
  • Полученный результат (для пожелания и вопроса) - описание фактического действия программы с указанием, чем оно не соответствует ожидаемому результату.
  • Порядок пошагового воспроизведения и полученный результат (для ошибки) – пошаговое описание выполняемых в программе действий.
  • Ожидаемый результат (для пожелания и ошибки) - подробное описание предполагаемого результата после выполнения вышеуказанных действий.
  • Что вызывает вопрос (для вопроса) – описание возникающих сложностей.

К форме может быть прикреплен файл для иллюстрации описываемой ситуации в программе.

После успешной отправки обращения ему присваивается номер. Если в списке "Мои обращения" номер пустой, значит в ближайшее время будет выполнена повторная попытка отправки обращения. После разбора обращения, свами свяжется консультант по указанным в обращении контактным данным.

Сообщение об ошибке может быть отправлено, если:

  • программный продукт зарегистрирован. Регистрацию можно выполнить из формы по кнопке Зарегистрировать продукт.
  • в программе включена настройка Разрешить автоматическую отправку сведений об использовании программы в фирму «1С-Рарус» (Администрирование – Интернет-поддержка и сервисы – Центр мониторинга). Данная настройка разрешает автоматически собирать статистику, необходимую разработчикам для анализа работы программы. При этом передача статистики не замедляет работу программы.
  • данное обращение ранее не отправлялось в службу поддержки (редактирование старых обращений недоступно).
  • в системе отключен механизм разделения данных.
