Документ "Начисление премий"

Документ предназначен для отражения премий сотрудников, не связанных с начислением зарплаты по документам Путевой лист, Ремонтный лист, Маршрутный лист.
Данные документа используются для заполнения начислений в документе Начисление зарплаты и могут быть выгружены в типовые конфигурации.

Реквизиты

    • Организация. Ссылается на справочник «Организации». Организация, по которой вводится документ. 
    • Подразделение. Ссылается на справочник «Подразделение организаций». Подразделение, по которому вводится документ.
    • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник, по которому выполняется начисление. 
    • Период начисления. Дата. Дата отражения начислений в учете. Может отличаться от даты документа.
    • План затрат. Ссылается на справочник «Планы применения затрат». Используется для получения Статья расхода для учета оперативных расходов.

      Табличная часть «Начисления»

      • Тариф. Ссылается на справочник «Тарифы зарплаты». Тариф зарплаты с применением "Премия", используемый для расчета премий. 
      • Начисление. Ссылается на план видов расчета «Начисления». Вид начисления со способом расчета "Фиксированной суммой" для учета начислений.
      • ТС. Ссылается на справочник «Транспортные средства и оборудование». Транспортное средство, с которым связано начисление премий.
      • Сумма. Сумма начисления в валюте управленческого учета. 

      Движения

      • Регистр расчета «Начисления». Выполняется начисление суммы премий сотрудника.
      • Регистр накопления «Оперативные расходы». Выполняется приход по статье расходов, указанной в Плане затрат документа.
