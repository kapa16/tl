Документ "Диспозиция ТС"

Назначение

Документ служит для регистрации диспозиции (местоположения и состояния) автомобилей.

Относится к подсистеме "Работа ТС".

Реквизиты шапки документа

  • ТС. Ссылается на справочник «Транспортные средства». Транспортное средство. Если в документе указано несколько транспортных средств, то в форме списка будут отображаться указанные ТС через запятую.
  • Состояние. Ссылается на справочник «Состояние ТС». Новое состояние ТС. Если в документе указано несколько различных состояний ТС, то в форме списка не будут отображаться указанные состояния, будет отображаться надпись "Несколько состояний".
  • Местоположение. Ссылается на справочники «Пункты назначения». Новое местоположение ТС.
  • Ответственный. Ссылается на справочник «Пользователи». Пользователь, ответственный за ввод информации в документ.
  • Контрагент. Ссылается на справочник «Контрагенты». Контрагент, связанный с диспозицией. Например, выполнение ездок для контрагента такого то.
  • Заказ. Ссылается на заказ, согласно которому изменяется диспозиция ТС.

Заполнение табличной части "ТС"

Табличная часть предназначена для отражения состояний нескольких ТС в одном документе.
