Документ "Реестр перевозок"

Документ предназначен для отражения перевозчиком факта оказания услуг по перевозкам по конкретному перевозчику.

Относится к подсистеме "Услуги".

Согласование документа

Реестр перевозок подключен к механизму согласования документов. После завершения процесса согласования реестр становится недоступен для редактирования внешними пользователями.

Заполнение табличной части "Услуги"

Табличная часть может быть заполнена автоматически за произвольно выбранный период времени или еще не выставленными услугами по кнопке "Заполнить". В табличную часть попадают данные регистров накопления Предоставленные услуги и Выставленные услуги, относящихся к указанному контрагенту/перевозчику и договору. Детальность заполнения табличной части определяется настройками Шаблона заполнения по услугам (Заполнить - Заполнить по услугам - Шаблон заполнения).
