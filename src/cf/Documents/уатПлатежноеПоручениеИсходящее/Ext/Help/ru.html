Документ «Платежное поручение исходящее»

Документ «Платежное поручение исходящее» предназначено для учета расхода безналичных денежных средств.

При расходовании денежных средств необходимо выбрать банковский счет, с которого будут списываться денежные средства. На счете денежные средства могут храниться в одной валюте. Поэтому валюта, указанная в счете определяет валюту оформляемого документа.

Реквизиты

  • Документ основание. В зависимости от значения в реквизите "Вид операции" используется при заполнении документа "Платежное поручение" в "Бухгалтерия предприятия 3.0". Если выбрано значение "Прочие расчеты с контрагентами" и документом основанием является документ "Штраф", то при выгрузке "Платежного поручения исходящего" в конфигурацию "1С:Бухгалтерия предприятия", из "Штрафа" будут заполнены реквизиты платежей в бюджет (КБК, ОКТМО и прочие). Флаг "Требуется указывать реквизиты платежа в бюджет" надо проставить вручную. Для отображения флага на форме в "Бухгалтерия предприятия 3.0" получатель должен являться государственный органом.

Отражение расходования по одному договору и сделке

При отражении поступления по одному договору и сделке в полях ввода формы документа необходимо указать:

  • сумму платежа (изменяется автоматически при редактировании суммы документа)
  • договор с контрагентом;
  • сделку (при необходимости);
  • курс валюты взаиморасчетов по договору, по которому будет учтен платеж во взаиморасчетах или сумму, на которую изменится состояние взаиморасчетов;

Возможно выбрать необходимый договор и сделку для погашения из списка текущих задолженностей по указанному контрагенту. Список открывается по нажатию кнопки «Подбор».

Отражение расходования по нескольким договорам и сделкам

В этом режиме становится доступной для редактирования табличная часть «Расшифровка платежа». Для каждой строки табличной части необходимо указать:

  • договор с контрагентом;
  • сделку (при необходимости);
  • сумму платежа (часть суммы документа, относимую на данный договор и сделку); итог по колонке «Сумма платежа» должен равняться сумме документа; 
  • курс валюты взаиморасчетов по договору, по которому будет учтен платеж во взаиморасчетах или сумму, на которую изменится состояние взаиморасчетов;

Возможно подобрать необходимые договоры и сделки для погашения из списка текущих задолженностей по указанному контрагенту. Список открывается по нажатию кнопки «Подбор». Также, можно автоматически заполнить табличную часть на основании информации по непогашенным задолженностям. Обработка для автоматического заполнения вызывается по нажатию кнопки «Заполнить» меню табличной части.

Разнесение платежей по Заказам на ТС

ТЧ отображается при включенной настройке организации "Разнесение платежей по Заказам на ТС". Для режима документа "Списком" набор строк ТЧ отображается по текущей строке ТЧ "Расшифровка платежа".
ТЧ доступна для редактирования при ведении взаиморасчетов "По счетам" и "По накладным".
ТЧ может быть заполнена:

  • автоматически при вводе на основании счетов и накладных, если в них указаны заказы;
  • автоматически при загрузке данных из типовой конфигурации при синхронизации данных, если включена настройка организации "Автоматическое разнесение платежей по Заказам на ТС при синхронизации данных";
  • с помощью формы "Подбор по задолженности";
  • ручным редактированием строк.
