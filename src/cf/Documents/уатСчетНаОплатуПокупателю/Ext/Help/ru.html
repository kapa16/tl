Документ "Счет на оплату покупателю"

Документ предназначен для отражения факта выставления счета покупателю. В документе указывается контрагент - покупатель услуг, банковский счет нашей организации, на который будут начислены денежные средства.

Относится к подсистеме "Услуги".

Заполнение табличной части "Услуги"

Табличная часть может быть заполнена автоматически за указанный период времени по кнопке "Заполнить". Подробно настройки заполнения см. в форме "Заполнение по услугам".

Заполнение табличной части "Услуги (регл.)"

Табличная часть предназначена для отражения услуг в регламентированном учете, если используется свертка (установлен флаг "Свертка услуг для отражения в регл. учете"). При этом данные используются при выводе документа на печать и обменах с типовыми конфигурациями 1С. Не редактируется. Заполняется по данным табличной части "Услуги (упр.)" при установке флага "Свертка услуг для отражения в регл. учете" автоматически или может быть заполнена вручную при нажатии на кнопку "Заполнить / Обновить по данным услуг (упр.)".

Платежный календарь

При записи документа, для всех связанных с ним Заказов на ТС, выполняется расчет Дат платежей по Платежному календарю при условии наступления событий согласно графика оплаты по договору. Дата платежа указывается вручную для этапов оплаты вида "Произвольная оплата" при ведении взаиморасчетов "По счетам" или "По заказам". Также она может быть указана, если график оплаты в договору не задан, при этом вся задолженность распределяется на предопределенный этап "Произвольная дата".

Вывод QR-кода на печать

При установке значения "Платежные реквизиты" в настройке "Вариант печати QR-кода" на вкладке "Дополнительно" при печати будет выводиться QR-код в шапке печатной формы, содержащий информацию о платеже. Значение настройки "Платежные реквизиты" недоступно, когда в документе не заполнен реквизит "Банковский счет". Если значение настройки не заполнено или установлено значение "Не печатать", тогда QR-код не будет выводиться при печати.

Ввод на основании

Документ может быть введен на основании документов "Реализация услуг", "Товарно-транспортный документ", "Заказ на ТС", "Маршрутный лист".
