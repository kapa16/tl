Документ "Начисление удержаний"

Документ предназначен для отражения удержаний сотрудников, связанных с порчей имущества, штрафов ДТП и т.д.
Данные документа используются для заполнения удержаний в документе Начисление зарплаты и могут быть выгружены в типовые конфигурации.

Реквизиты

  • Организация. Ссылается на справочник «Организации». Организация, по которой вводится документ. 
  • Подразделение. Ссылается на справочник «Подразделение организаций». Подразделение, по которому вводится документ.
    •
  • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник, по которому выполняется удержание. 
  • Период удержания. Дата. Дата отражения удержания в учете. Может отличаться от даты документа.

Табличная часть «Удержания»

  • Тариф. Ссылается на справочник «Тарифы зарплаты». Тариф зарплаты с применением "Удержание", используемый для расчета удержания. 
  • Удержание. Ссылается на план видов расчета «Удержания». Вид удержания со способом расчета "Фиксированной суммой" для учета удержаний.
  • ТС. Ссылается на справочник «Транспортные средства и оборудование». Транспортное средство, с которым связано удержание.
  • Сумма. Сумма удержания в валюте управленческого учета. 
  • Основание. Основание удержания - Штраф.

Заполнение по штрафам

Табличную часть документа можно заполнять по данным документов Штраф. В форме заполнения нужно указать период заполнения и действие при создании документов (если заполняется из списка документов), отборы по организации, сотруднику и другие. Выполняется проверка наличия уже созданных удержаний по штрафам, если они уже введены, то новые удержания не создаются.

Движения

  • Регистр расчета «Удержания». Выполняется начисление суммы удержания сотрудника.
  • Регистр накопления «Оперативные доходы». Выполняется приход по статье доходов, указанной в Удержании в табличной части Удержания.
