Документ "Реестр оплаты штрафов"

Назначение

Документ предназначен для оплаты штрафов в автоматическом режиме по внесенному авансу. Для выполнения платежей необходимо предварительно перечислить Аванс в пользу сервиса. 

При проведении документа, если поле ID реестра не заполнено, реестр на оплату штрафов будет добавлен в сервис. После успешного создания реестра в сервисе штрафов в поле ID реестра подставится полученный идентификатор нового реестра. 

При пометке на удаление документа, если поле ID реестра заполнено, реестр на оплату штрафов будет удален из сервиса.

После создания реестра в сервисе штрафов необходимо некоторое время, чтобы получить актуальные платежные реквизиты, проверить постановления на возможность оплаты. Обычно это занимает 1-5 минут (в зависимости от количества штрафов).

 

Реквизиты

  • Организация. Ссылается на справочник «Организации». 
  • Сумма всех постановлений. Тип значения: «Число».  Сумма всех оплачиваемых постановлений.  
  • Сумма комиссии. Тип значения: «Число».  Комиссия сервиса за оплату постановлений за клиента.  
  • Сумма с учетом комиссии. Тип значения: «Число».  Итоговая стоимость включающая комиссию сервиса штрафов.  
  • Дата создания реестра. Тип значения: «Дата». Дата и время создания реестра в сервисе штрафов. 
  • ID реестра. Тип значения: «Строка». Идентификатор реестра в сервисе штрафов. 
  • Ответственный. Ссылается на справочник «Пользователи». 

 

Относится к подсистеме "БДД".
