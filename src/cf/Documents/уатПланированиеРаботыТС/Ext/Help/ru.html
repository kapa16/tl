Документ "Планирование работы ТС"

Назначение

Документ предназначен для отражения планов по предполагаемым объемам работы ТС и планов оказания предоставленных услуг. 

Относится к подсистеме "Транспорт".

Реквизиты

  • Сценарий. Реквизит определяет:
    • раздел планирования - "Выработка ТС", "Услуги (Выручка)", "Доходы и Расходы" или "Объемы перевозок";
    • периодичность планирования - год, месяц и т.д.;
    • состав колонок в табличной части "Состав плана".
  • Дата планирования. Отображает дату начала периода планирования.

Состав плана

Состав плановых данных вводится в табличную часть документа.  По каждому объекту указывается плановый объем работы ТС в количественной ("Выработка ТС" или "Объемы перевозок") или суммовой ("Услуги (Выручка)" или "Доходы и Расходы") оценке. Для раздела планирования "Объемы перевозок" также возможно указание оценки в процентах от общего количества выработки, в зависимости от настроек сценария.

Признак "Ручное редактирование" отображается только в тех строках, в которых выбраны рассчитываемые по формуле показатели. При изменении значения по такому показателю признак "Ручное редактирование" устанавливается в Истина. Признак используется, чтобы сохранения информации о ручном изменении значения.

Кнопка "Рассчитать по формуле" используется для автоматического расчета всех показателей, значения которых рассчитываются по формулам. Относится только к пользовательским параметрам выработки для которых способ расчета установлен в "По заказам на ТС и МЛ".

В случае, когда выбран сценарий по разделу планирования "Объемы перевозок", доступны колонки "Запланировано" и "Распределено". Значения в этих колонках заполняются в разрезе наборов аналитик - Маршрута, Направления перевозки, Объекта строительства и т.д - определенных в Сценарии.

Значение в колонке "Распределено" не редактируется, оно рассчитывается автоматически как сумма по всем перевозчикам и их договорам для данного набора аналитик. В корректно заполненном документе значение в колонке "Запланировано" должно совпадать с "Распределено".

Заполнение

Состав документа может быть заполнен в автоматическом режиме, для этого используется помощник планирования, вызываемый по кнопке «Заполнить».
