Документ "Взаимозачет"

Документ предназначен для отражения в учете операций по взаимозачетам между двумя контрагентами, имеющими дебиторскую и кредиторскую задолженность, или для перераспределения задолженностей по различным договорам и сделкам.   

Относится к подсистеме "Денежные средства".

Реквизиты

  • Организация. Тип значения: справочник «Организации». Организация, по которой вводится взаимозачет. Может отличаться от организации, к которой относится ТС на момент проведения документа.
  • Подразделение. Тип значения: справочник «Подразделение организаций». Подразделение, по которому вводится взаимозачет.  Может отличаться от подразделения, к которому относится ТС на момент проведения документа.
  • Дебитор. Ссылается на справочник «Контрагенты». Контрагент, который имеет долги перед нашей компанией.
  • Кредитор. Ссылается на справочник «Контрагенты». Контрагент, которому должна наша компания.
  • Ответственный. Пользователь, ответственный за ввод информации в документ.

Табличная часть «Состав»

  • Договор дебитора, Договор кредитора. Ссылается на справочник Договоры контрагентов. Два договора взаиморасчетов: дебитора и кредитора.
  • Сделка дебитора, Сделка кредитора. Ссылается на документы-сделки. Сделки, по которым возникли задолженности дебитора и кредитора. 
  • Валюта, курс, кратность. Валюта, курс, кратность договора дебитора/кредитора. Валюта соответствует валюте договора дебитора/кредитора. Курс и кратность можно изменить вручную при необходимости.
  • Сумма долга дебитора. Сумма взаимозачета в валюте договора дебитора. Существующая задолженность дебитора перед компанией будет уменьшена на указанную сумму.
  • Сумма долга кредитора. Сумма взаимозачета в валюте договора кредитора. Существующая задолженность кредитора перед компанией будет увеличена на указанную сумму.

Платежный календарь

1) Документ уменьшает (закрывает) задолженность по Платежному календарю по Дебитору. Закрытие задолженности выполняется по этапам оплаты и заказам на ТС по FIFO - сначала закрывается более ранняя. 
2) Документ увеличивает задолженность по Платежному календарю по Кредитору. При этом заново выполняется распределение Суммы кредитора по настройкам и графику оплаты Договора кредитора.
3) Есть возможность указания Сделки кредитора равной Сделке дебитора. В этом случае Договор кредитора не будет соответствовать Сделке кредитора.
При ведении взаиморасчетов "По Заказам" в договорах дебитора и кредитора и выборе сделки кредитора равной сделке дебитора, плановые даты платежей дебитора сохранятся и для кредитора, при условии полного закрытия задолженности и идентичности графика оплаты в договорах дебитора и кредитора.
