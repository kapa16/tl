Справочник "Модели агрегатов"

Назначение

Справочник предназначен для классификации агрегатов (шины, аккумуляторы, аптечки, огнетушители) по их техническим характеристикам.

Относится к подсистеме "Ремонты и агрегаты".

Реквизиты

  • Тип агрегата. Для модели необходимо указать тип агрегата. В зависимости от типа становятся доступны дополнительные параметры.
  • Производитель.  Производитель агрегата.  
  • Параметр выработки. Параметр выработки агрегата, по которому производится расчет выработки в путевых листах и других документах. Если параметр выработки указан, то при проведении документов будут формироваться движения по регистру "Выработка агрегатов" и обороты будет возможно анализировать в отчетах.
    Для шин параметр выработки должен использоваться только, когда нужно вести учет одновременно для нескольких параметров выработки. Для шин нельзя выбирать параметр выработки, который измеряется в той же единице измерения, что и единица измерения износа.
    Ссылается на справочник «Параметры выработки ТС и водителей».
  • Параметры износа. Реквизиты, влияющие на учет износа по агрегатам
    • начисление износа - вариант начисления износа. 
      1) Выработка. Износ начисляется Путевыми листами и Технологическими путевыми листами по выработке ТС, на которое установлен агрегат. Выработка берется по параметру выработки для начисления износа.
      2) Период. Износ начисляется документом "Прочие доходы и расходы" (операция "Расход") один раз в месяц по норме затрат от первоначальной стоимости агрегата.
      3) Не начислять. Износ не рассчитывается и не начисляется.
    • норма выработки - норма в км или в ч до полного износа;
    • параметр выработки - параметр выработки для начисления износа. Пробег общий, время в работе - основные используемые параметры выработки для учета пробега шин и времени работы агрегата.
    • срок службы, месяцев;
    • норма затрат, % от стоимости на 1000 км или на 1000 ч - затраты на шины в % от стоимости агрегата;


Для шин дополнительно указываются:

  • модель;
  • размерные характеристики шин;
  • сезонность;
  • единица измерения износа - километры или часы. Если выбраны километры, то пробег шин рассчитывается по параметру выработки "Пробег общий", если часы, то по параметру выработки "Время в работе";
  • индекс скорости шин;
  • индекс нагрузки шин;
  • тип конструкции шин;
  • дополнительная информация.  


Для аккумуляторов дополнительно указывается:

  • емкость аккумулятора;
