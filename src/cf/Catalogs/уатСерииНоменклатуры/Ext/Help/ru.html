Справочник "Серии номенклатуры (агрегаты)"

Назначение

Данный справочник предназначен для ведения учета шин и аккумуляторов.

Информация о введенных в справочник значениях предоставляется пользователю для выбора данных о серии номенклатуры при вводе шины, аккумулятора или других агрегатов в таких документах, как «Ввод начальных данных по агрегатам», «Поступление агрегатов на склад», «Ремонтный лист».

Для агрегатов типа "Шина" предусмотрена возможность учета пробега и износа. Для расчета износа необходимо настроить параметры износа в справочнике  «Модели агрегатов».

Ввод первоначальных данных по установленным агрегатам на автомобилях и остаткам на складах оформляется документом «Ввод начальных данных по агрегатам». Дальнейшее поступление новых агрегатов на склад оформляется документом «Поступление агрегатов на склад».

Установка и снятие шин, аккумуляторов и прочих агрегатов с/на автомобиль оформляется документом «Ремонтный лист».

Установка и снятие прочих агрегатов оформляется документом «Операции с агрегатами».

Относится к подсистемам "Управление ремонтами".

Реквизиты

  • Номенклатура. Номенклатура агрегата. Ссылается на справочник «Номенклатура»;
  • Серийный номер. Серийный номер агрегата.
  • Модель. Модель агрегата. Ссылается на справочник «Модели агрегатов»;
  • Тип агрегата. Тип агрегата. В случае если указана модель, то поле заполняется из модели и недоступно для редактирования.  ссылается на справочник «Типы агрегатов»;
  • Начальная стоимость. Первоначальная стоимость агрегата.
  • Валюта. Валюта первоначальной стоимости агрегата. Если не указана, то будет использована валюта регламентированного учета.
  • Начало эксплуатации. Тип значения: «Дата». Начало эксплуатации аккумулятора. Используется только для аккумуляторов.
  • Годен до. Тип значения: «Дата». Срок годности аккумулятора. Используется только для аккумуляторов.
  • Дата производства. Тип значения: «Дата». Дата производства шины в формате "номер недели в году / год". Используется только для шин.
  • Срок службы. Тип значения: «Число». Срок службы шины, месяцев.  Используется только для шин.
  • Учет износа. Реквизиты, влияющие на учет износа по агрегатам
    • Параметр выработки - параметр выработки для начисления износа. Пробег общий, время в работе - основные используемые параметры выработки для учета пробега шин и времени работы агрегата.
    • Текущая выработка - общая выработка агрегата за весь период учета в единице параметра выработки.
    • Степень износа, % - фактическая измеренная степень износа шины;
    • Расчетная степень износа, % - степень износа по данным учета, это отношение текущей выработки агрегата к норме выработки по модели.
  • Учет выработки
    • Параметр выработки. Параметр выработки агрегата, по которому производится расчет выработки в путевых листах и других документах. Если параметр выработки указан, то при проведении документов будут формироваться движения по регистру "Выработка агрегатов" и обороты будет возможно анализировать в отчетах. 
      Для шин параметр выработки должен использоваться только, когда нужно вести учет одновременно для нескольких параметров выработки. Для шин нельзя выбирать параметр выработки, который измеряется в той же единице измерения, что и единица измерения износа.
      Ссылается на справочник «Параметры выработки ТС и водителей»; 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.

Способ вызова

Вызывается из меню "Складской учет --> Серии агрегатов".
