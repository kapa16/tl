Справочник "Классификаторы категорий ТС ЕЭК"

Назначение

Справочник содержит список используемых категорий транспортных средств.

Относится к подсистеме "Транспорт".

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 50.

Реквизиты

  • Описание. Строка. Содержит описание категории ТС.
  • Код вида паспорта ТС. Строка. Содержит код вида транспортного средства.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов. Содержит команду создания по классификатору 
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
