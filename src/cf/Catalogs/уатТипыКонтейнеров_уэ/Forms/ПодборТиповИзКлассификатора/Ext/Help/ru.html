Содержит общую информацию о типах контейнеров и предназначен для добавления новых типов контейнеров

Добавление типа контейнера из классификатора

  • Выделив один или несколько типов контейнеров, нажмите кнопку Выбрать.
  • Воспользуйтесь двойным щелчком мыши по строке для добавления типа контейнера в список типов контейнеров.
