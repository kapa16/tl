Справочник "Виды обслуживания ТС"

Назначение

Справочник "Виды обслуживания ТС" предназначен для классификации работ при проведении ТО, планово-профилактических работ, ремонтов и т.д.

Относится к подсистеме "Ремонты и агрегаты".

Данные этого справочника используются в документах "Заявка на ремонт", "Ремонтный лист" и в справочнике "Модели ТС и оборудования" при настройке периодичности прохождения планового технического обслуживания. 

Реквизиты

  • Вид обслуживания ТС. Вид обслуживания/ремонта ТС:
    • текущий ремонт,
    • средний ремонт,
    • капитальный ремонт,
    • дооборудование,
    • модернизация и т.п.
  • Цвет. Выбранное значение используется при отображении вида обслуживания в отчете "График ТО".
  • Планирование ТО. Флажок должен быть установлен для того, чтобы вид обслуживания был доступен для использования в плановом ТО.
  • Циклический. Если в карточке вида обслуживания установлен данный флажок, то для данного вида ТО цикличность проведения настраивается для моделей транспортных средств по дате или пробегу, ТО может проводиться неограниченное число раз до окончания эксплуатации ТС. Если флажок «Циклический» не установлен, ТО проводится по сервисной книжке (имеет план). В этом случае для определенной модели в сервисной книжке для каждого ТО заранее известны:
    • перечень работ, которые должны быть выполнены при проведении ТО;
    • перечень и количество запасных частей и материалов, которые должны быть заменены на ТС.

         При проведении ТО по сервисной книжке для ТС указывается цепочка последовательных ТО, каждое из которых выполняется 1 раз.

    • Мобильное приложение. Признак доступности вида обслуживания ТС для выбора в мобильном приложении "1С-Рарус: Водитель УАТ".
    • Не отражается в учете. При установленном флаге Ремонтные листы с этим видом обслуживания не выполняют движения по партиям и взаиморасчетам и не выгружаются в типовые конфигурации при синхронизации данных.

    Табличные части

    • Работы. Здесь отображаются связанные работы из регистра Сервисная книжка
    • Запчасти. Здесь отображаются связанные запчасти из регистра Сервисная книжка
    • Замещаемые работы/запчасти. Здесь можно указать замещаемый вид обслуживания ТС и выбрать одну или несколько работ/запчастей, которые будут замещаться при планировании требуемой номенклатуры в документе Планирование работы ТС и в Помощнике планирования работы ТС. В этом случае, когда для одного ТС будут планироваться на одну и ту же дату замещаемый и замещающий виды обслуживания, для замещаемого вида обслуживания не будет планироваться замещаемые работы и запчасти.
