Редактирование макета

Служит для визуального представления пользователю внешнего вида макета (с помощью табличного документа) и списка доступных полей для редактирования (с помощью табличного поля). С помощью данной формы пользователь может инициировать добавление, редактирование и удаление полей, очистку макета, а также сохранение описания макета в XML-файл и загрузку ранее сохраненных XML-файлов.

Редактор этикеток открывается по ссылке Администрирование - Настройка оборудования - Шаблоны этикеток.

После создания этикетки можно приступить к редактированию макета.

Начать редактирование макета.

Редактор этикеток представляет из себя форму со следующими реквизитами:

1) Ширина - ширина этикетки в миллиметрах;

2) Высота - высота этикетки в миллиметрах;

3) Стандартные размеры - список для выбора размера этикетки из списка стандартных;

4) Ориентация - ориентация этикетки (поворот этикетки против часовой стрелки перед передачей в печать)

5) Макет - поле табличного документа для визуального представления внешнего вида этикетки. Высота каждой строки и ширина каждого столбца равны одному миллиметру.

6) Поля - табличная часть, содержащая список полей текущего макета этикетки.

При помощи данной формы можно инициировать добавление/редактирование и удаление поля, а также полную очистку макета этикетки, для этого служат соответствующие кнопки на командной панели табличной части "Поля". Для увеличения или уменьшения масштаба табличного документа воспользуйтесь сочетанием Ctrl+Колесо мыши.

Также в меню дополнительных функций доступна выгрузка/загрузка макетов этикеток из XML-файлов.
