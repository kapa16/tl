Редактирование поля макета

Служит для указания значений свойств конкретного поля. При переходе к добавлению или редактированию поля макета этикетки будет открыта форма редактирования поля, реквизиты которой можно разделить на три группы: общие, частные, индивидуальные (только для одного из типов поля).

В общие реквизиты входят:

1) Тип поля - тип содержимого поля (текст, штрихкод, картинка или дополнительно (пользовательские данные)). В зависимости от значения реквизита “Тип поля” состав доступных реквизитов меняется; 

2) Тип заполнения - тип заполнения поля данными ("Параметр" - данные поля конкретной единицы этикетки задается значением заполнения, "Постоянное значение" - значение одинаково для всех этикеток)

3) Значение - имя одного из доступных полей или строковая константа, в зависимости от значения реквизита "Тип заполнения".
В случае, когда значение реквизита "Тип заполнения" равен "Параметр", для поля "Значение" становится доступной кнопка выбора параметра из списка, для чего открывается соответствующая форма.

Для типа поля "Штрихкод" следует выбрать значение "Штрихкод" (данные регистра Штрихкоды). Если значение штрихкода хранится в справочнике Номенклатура, следует указать соответствующий реквизит справочника.

4) Значение по умолчанию - значение, которое будет подставлено в качестве значения поля, если оно не было указано.

Две остальные группы реквизитов влияют на визуальное представление этикетки, следовательно, не относятся к полям с типом "Дополнительно".

Следующие реквизиты также относятся к частным:

1) Лево - порядковый номер левого столбца, с которого начинается поле, относительно левого края этикетки. По сути, это отступ слева в мм.-1 мм;

2) Верх - порядковый номер верхней строки, с которой начинается поле, относительно верхнего края этикетки. По сути, это отступ сверху в мм. -1 мм;

3) Право - порядковый номер последнего столбца, в котором заканчивается поле, относительно левого края этикетки;

4) Низ - порядковый номер последней строки, в которой заканчивается поле, относительно верхнего края этикетки;

5) Ориентация - поворот содержимого поля против часовой стрелки. Обработка данного реквизита подобна свойству поворот текста в MS Excel.

6) Рамка (не поддерживается для типа поля "Штрихкод"):

a) Сверху - наличие рамки сверху;

b) Слева - наличие рамки слева;

c) Снизу - наличие рамки снизу;

d) Справа - наличие рамки справа;

e) Тип рамки - тип рамки (точечная, сплошная, двойная, пунктир);

f) Толщина рамки.

Состав индивидуальных реквизитов зависит от типа поля.

В случае "Текст":

· Шрифт - имя шрифта, размер, стиль и т.д.;

· Многострочность - флаг многострочности текста (аналогично переносу по словам);

· Горизонтальное положение - выравнивание текста по горизонтали (лево, центр, право);

· Вертикальное положение - выравнивание текста по вертикали (верх, центр, низ);

В случае "Штрихкод":

· Тип штрихкода - тип штрихкода (EAN8, EAN13 и т.д);

· Наличие подписи к штрихкоду - наличие под штрихкодом строки-подписи так называемой "HRI" (Human Readable Interpretation);

· Выводить контрольный символ - флаг, отвечающий за вывод в подписи к штрихкоду контрольного символа

· Размер шрифта подписи - собственно, размер шрифта подписи к штрихкоду.
